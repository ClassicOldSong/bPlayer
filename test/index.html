bPlayer - 原生音频播放器的绝佳替代品

#bplayer1/bp1 with autoplay enabled

#bplayer2/bp2 with loop enabled

#bplayer3/bp3 created from JSON with slim mode enabled

#bplayer4 converted from Audio element with loop enabled

#bplayer5 converted from Audio element with slim mode enabled

可以尝试打开控制台对 bp1, bp2, bp3 进行控制，例如：bp1.color("#FF2870");

对于由Audio转换而来的元素,可以直接根据原来的id获取元素进行控制，例如：
document.querySelector('#bplayer4').title = "whatever you want"
